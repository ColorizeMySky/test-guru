# Сущности системы TestGuru

1. **Пользователь**
   - Администратор
   - Преподаватель (автор тестов)
   - Ученик (тестируемый)

2. **Категория**
   - Название
   - Описание (опционально)

3. **Тест**
   - Название
   - Уровень сложности
   - Категория
   - Таймер (ограничение по времени)
   - Статус (черновик/опубликован)

4. **Вопрос**
   - Текст вопроса
   - Тип вопроса:
     - Одиночный выбор
     - Множественный выбор
     - Открытый ответ
   - Баллы за правильный ответ

5. **Ответ**
   - Текст ответа
   - Флаг правильности

6. **Результат теста**
   - Пройден/не пройден
   - Количество баллов
   - Процент правильных ответов
   - Время прохождения
